# Resume RAG Chatbot

An intelligent chatbot that answers natural language questions about your resumes using Retrieval-Augmented Generation (RAG). Built with **Streamlit**, **Google Gemini**, and **FAISS** – runs entirely in the cloud or locally.

## Features

- **Semantic search** over multiple resume files (`.txt`)
- **Google Gemini** for accurate, context‑aware answers
- **Clean, conversational UI** with an animated avatar
- **Fast and lightweight** – uses FAISS instead of ChromaDB for compatibility
- **Secure API key management** via Streamlit Secrets
